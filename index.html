The Hack Project

This project is designed for non-malicious purposes. It’s quite useful for discovering what’s on your network. You can use nmap, arp, ping, dig and host all in one place without needing to type the commands yourself. Its main purpose is for network owners who like monitoring their networks.
Thanks to SVGRepo for some of the icons used on this site.

Installation

To install the hack project, run the following command in your terminal: 
        
            /bin/bash -c "$(curl -fsSl https://raw.githubusercontent.com/myshy13/hackProject/refs/heads/main/hackInstaller)"

Usage

After installation, you can run the following command to start the program: 

        hack

After installation, you can run the following command to update the program: 

      hackUpdater
